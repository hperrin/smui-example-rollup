# SMUI Rollup Example

An example project implementing [Svelte Material UI](https://github.com/hperrin/svelte-material-ui) with [Rollup](https://github.com/rollup/rollup). (Using the Advanced Styling Method.)

# Get it Running

1. Clone the repo.
2. `npm install`
3. `npm run build`
4. `npm run serve`
5. http://localhost:5000
